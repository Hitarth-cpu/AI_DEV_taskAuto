# TaskAuto CLI Daemon

Persistent WebSocket daemon for [TaskAuto](https://github.com/Hitarth-cpu/AI_DEV_taskAuto).

## Setup

Set these secrets in the Space settings:

| Secret | Value |
|--------|-------|
| `DAEMON_TOKEN` | `hf_agent` (or any unique name) |
| `BACKEND_SERVER` | `ai-dev-taskauto.onrender.com` |
